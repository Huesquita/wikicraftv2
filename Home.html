¿What is minecraft?

Minecraft is a sandbox video game developed by Mojang Studios. The game was created by Markus "Notch" Persson in the Java programming language. There isn’t a main objective in Minecraft, but there is a certain flow of gameplay that is recommended for everybody. You start in this huge open world with nothing, and must gather materials from scratch to build yourself a shelter, get food, build a bed and survive until the next day.
